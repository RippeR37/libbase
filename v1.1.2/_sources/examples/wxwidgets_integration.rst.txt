wxWidgets integration module example
====================================

Below you can find source code of a ``wxwidgets_integration`` example that
showcases usage of the ``libbase`` integration module for ``wxWidgets`` library.
This example is located in the ``/examples/wxwidgets_integration/`` directory in
the repository and - if :ref:`enabled <configuration-libbase-build-examples>` -
will be built along the ``libbase`` library itself.
